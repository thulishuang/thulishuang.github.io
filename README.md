# Shuang Li Homepage

Personal academic homepage for Shuang Li.

Live site: https://thulishuang.github.io/

## Stack

- Jekyll
- Academic Pages / Minimal Mistakes based theme
- GitHub Pages deployment

## Main Content Files

- `_pages/about.md`: homepage content
- `_publications/`: publication entries
- `_data/navigation.yml`: top navigation
- `_config.yml`: site configuration
- `images/`: profile photo, logos, favicon assets

## Local Development

```bash
bundle install
bundle exec jekyll serve -l -H localhost
```

Then open `http://localhost:4000`.

## Notes

- The repository has been cleaned up from most unused Academic Pages sample files.
- Changes pushed to `main` are deployed by GitHub Pages automatically.
